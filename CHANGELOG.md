# Changelog

## 1.1.0

- Adds a Ritual Sites filter button to the world map using the modern HandyNotes map button column style.
- Adds category filters for seaweed/soggy nest markers and void-touched lynx thicket markers.
- Adds English and Simplified Chinese localization for marker text, settings, and the map filter menu.

## 1.0.0

- Initial release for World of Warcraft Retail 12.0.5.
- Adds HandyNotes markers for Ritual Site map 2594 only.
- Marks 8 Washed-Up Seaweed locations, the Soggy Nest at 30.03, 63.12, and 9 Rustling Hidden Thicket locations.
- Supports in-game waypoint creation with left-click and TomTom waypoints with right-click when TomTom is installed.
